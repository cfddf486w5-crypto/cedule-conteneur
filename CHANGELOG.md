# Changelog

## [1.0.0] - 2026-02-24
### Added
- Initial release of `DL_RPG_Idle_V1`.
- SwiftUI MVVM app architecture.
- Combat engine with auto-attacks and Power Strike skill.
- Zone progression (Forest -> Ruins), stage/boss flow.
- Loot generation, inventory, equipment, and selling.
- Hero leveling, XP scaling, and talent allocation/unallocation.
- Shop, settings, reset-save handling, and debug panel.
- Persistence via `UserDefaults` with corruption fallback.
